# Expense-Tracker 

A desktop budget tracker application built with Python and Tkinter.

## Features
- Add/Delete expenses
- Monthly and category reports
- Pie chart and bar chart visualizations
- Local CSV storage

## Technologies
- Python 3.x
- Tkinter
- Matplotlib
- CSV

## Installation
1. Clone repository
2. Run: `python budget_tracker.py`
